Ringkasan Materi: Form HTML

Tujuan: Santri memahami cara membuat form dan mengirim data.

1. Apa itu Form HTML?

Form HTML digunakan untuk mengambil input dari pengguna, seperti nama, email, pesan, dan lainnya.

2. Struktur Dasar Form

<form action="tujuan_data" method="post">
    <input type="text" name="nama">
    <button type="submit">Kirim</button>
</form>

action → tempat data dikirim
method → cara kirim data (GET / POST)

3. Jenis Input Form

  • text → teks biasa
  • email → email
  • number → angka
  • radio → pilih satu
  • checkbox → pilih banyak
  • textarea → pesan panjang

4. Atribut Penting

  • name → kunci data (WAJIB)
  • required → wajib diisi
  • placeholder → contoh isian

5. Contoh Form Kontak

<form action="#" method="post">
    <input type="text" name="nama" placeholder="Nama">
    <input type="email" name="email" placeholder="Email">
    <textarea name="pesan"></textarea>
    <button type="submit">Kirim</button>
</form>

6. Alur Pengiriman Data

  1. Pengguna isi form
  2. Klik tombol kirim
  3. Data dikirim ke action
  4. Data diproses (PHP / Apps Script)
  5. Data disimpan (Database / Spreadsheet)

7. Contoh Tujuan Data

  • Backend PHP → proses.php
  • Google Spreadsheet → Google Apps Script
