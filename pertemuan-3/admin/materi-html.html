Ringkasan Tag HTML Berdasarkan Fungsinya

HTML (HyperText Markup Language) digunakan untuk membuat struktur dan makna konten pada halaman web.

1. Tag Struktur Dasar

  • <html> : Pembungkus seluruh halaman
  • <head> : Informasi halaman
  • <body> : Konten yang ditampilkan

2. Tag Teks

  • <h1> - <h6> : Judul
  • <p> : Paragraf
  • <br> : Ganti baris
  • <strong> : Teks penting
  • <em> : Teks miring

3. Tag Link & Media

  • <a> : Link
  • <img> : Gambar
  • <audio> : Audio
  • <video> : Video

4. Tag List

  • <ul> : List tidak berurutan
  • <ol> : List berurutan
  • <li> : Item list

5. Tag Tabel

  • <table> : Tabel
  • <tr> : Baris
  • <th> : Judul kolom
  • <td> : Isi kolom

6. Tag Form

  • <form> : Form input
  • <input> : Input data
  • <textarea> : Teks panjang
  • <select> : Pilihan
  • <button> : Tombol

7. Tag Semantic (Layout)

  • <header> : Bagian atas
  • <nav> : Navigasi
  • <section> : Bagian konten
  • <article> : Konten mandiri
  • <footer> : Bagian bawah

8. Tag Container

  • <div> : Pembungkus blok
  • <span> : Pembungkus inline

Catatan:
HTML untuk struktur,
CSS untuk tampilan,
JavaScript untuk interaksi.
